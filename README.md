# 飞行雪绒桌面伴侣
## 安装（用户）
前往这个窗口右侧 release，下载最新版本 dmg，打开并安装
注意飞行雪绒会要求你开启 MacOS 的录屏/terminal 输入功能，请帮她打开，谢谢。

准备一个 DeepSeek API Key（大约 10 块钱能玩很久了）。第一次使用飞讯或更改key，右键点击飞行雪绒，输入，自动保存为该应用容器中的环境变量


## 安装（开发者）
首先在你的根目录中设置：
|-resources     用于存放多媒体资源
|
|-src           存放这个仓库里的代码
|
__release       可以不手动创建，会随build脚本src/release_macos.sh自动更新

我使用uv进行python环境管理，因此首先在你的base环境中：
```bash
    pip install uv
```
然后使用 uv 进行虚拟环境管理并运行 debug 模式
```bash
    cd src
    uv sync 
    uv add ****
    uv run main.py
```
记得准备好你的 DeepSeek API Key。https://platform.deepseek.com/
第一次使用飞讯或更改key，右键点击飞行雪绒，输入，自动保存为该应用容器中的环境变量

⚠️Warning：我做了大量的容器隔离，但我的屎山仍然有可能污染您的开发环境，请您注意防范。

## 功能
### 基本移动
飞行雪绒会在你的屏幕上飞行或闪现，你能跟上她吗？
鼠标左键按住拖动可以把她拽回来。
### 鼠标左键点击：
`打开飞讯`：进入与飞行雪绒的聊天页面（DeepSeek + 人设 json文件）。⚠️Out of character 警告。建议先“热身几轮”，注入一点人设知识。
`爱弥斯，变身！`：播放与游戏中相同的待机动画结束界面。⚠️有泪点，注意防范。抠图不是很干净，作者准备有空换一个源。
`电子幽灵登场！`：启动你的终端。与爱弥斯一起写命令行吧！。
`召唤雪绒海豹/拜拜海豹`：随机散落 1～12 只爱弥斯最喜欢的雪绒海豹在你的屏幕上。点击可消除（类似打地鼠）。点击“拜拜海豹”可清空海豹。
`你看，又唱`：启动飞行雪绒电台。可导入本地音频。最小化时会有一个浮动条。关闭时自动结束播放器进程。⚠️注意防范《远航星的告别》和《纸飞机》。
`拜拜`:退出应用，下次再见。
### 鼠标右键点击：
`设置窗口`，目前只有 API Key 的输入。
